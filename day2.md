## Project

### Casey Quest

Player: Champ 

Enemy: Dragon

Goal: Casey 

Objective:
Reach Casey before the timer runs out while avoiding the Dragon.

## Features Implemented

* Arrow key movement
* Countdown timer
* Win condition
* Lose condition
* Collision detection
* Custom block for player reset
* Multiple sprites

## Challenges

* Figuring out which Scratch blocks to use
* Converting the game idea into working logic
* Understanding how conditions and loops work together
